# 隐私与 Cookie

- Cookie 属于账号凭据。
- 不要让用户在聊天中发送小红书密码、短信验证码、原始 Cookie、Token 或完整账号凭据。
- `xhs login` 读取的是用户本机浏览器登录态；如果它调起本机浏览器窗口，用户可以在窗口内完成网页登录授权。
- 浏览器窗口内输入密码或扫码确认属于本机登录流程，不应出现在最终报告、日志摘要或 Skill 输出中。
- 不要在最终报告中打印原始 Cookie、Token、`xsec_token`、完整账号 ID 或作者账号隐私信息。
- 不要把 Cookie 提交到项目仓库。
- 登录状态、Cookie 和缓存由上游 `xhs-cli` 在用户本机管理。
- 本技能默认只搜索、阅读和整理笔记，不发帖、不评论、不点赞、不收藏、不关注。
- 如果用户明确要求写操作，先说明风险，再确认是否继续。
- 如果小红书触发风控、验证码或 IP 风险，停止高频请求，降低频率，并把限制说明给用户。
